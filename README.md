## Sobre

Este repositório tem como objetivo armazenar as atividades avaliativas desenvolvidas na disciplina de "Desenvolvimento de Interfaces para Web", do UniSenac de Porto Alegre/RS.

![Logo](https://api.senacrs.com.br/bff/site-senac/v1/file/c9a87f58237d68bcfeb49fefd036eeeb7a0bd7.png)

## Conteúdo

Foram abordados os principais fundamentos de HTML e CSS, tais como:

### HTML
- Tabelas
- Listas ordenadas e não ordenadas
- Formulários
- Organização semântica

### CSS
- Estilização
- `width` e `height`
- `margin` e `padding`
